# VideospielShop

Einrichtung:

1. SQL-Create file ausführen

2. Datenbankverbindung in VideospielshopDBConnection auf... ändern

    hostname='localhost'
    dbname='videospielshop' 
    Benutzername: 'root'
    Passwort: ''
     
3. Index.php starten (Chrome)
 
4. Admin-Benutzer erstellen unter Registrieren: Wichtig: Email = root@root.de, den rest nach belieben

5. Als Admin-Benutzer einloggen->

5.5 Videospiele bearbeiten-/ und Kunden bearbeiten-Funktion auf index.php freigeschaltet

6. Auf die jeweilige neue Funktion clicken

7. Autogenerierung von Videospielen: 
      7.1 click auf "NEU"
      7.2 bei Plattform "666" eingeben
      7.3 click auf "SPEICHERN"
      7.4 generiert nun default Spiele 

8. Autogenerierung von Kunden: 
      8.1 click auf "NEU"
      8.2 bei Vorname "666" eingeben
      8.3 click auf "SPEICHERN"
      8.4 generiert nun default Kunden (die jeweiligen passwörter sind immer der Vorname in klein, z.B. bei "marc.seiler@siemens-energy.com" ist das Passwort: "marc")

9. Website eingerichtet!

10. Viel Spaß...

Tipps:

Das Kaufen von Videospiele funktioniert nur wenn man eingeloggt ist und dann auf das jeweilige "Kaufen: 39.99€" clickt
Beim click auf seine Email-Adresse auf der Startseite wird man zum Profil weitergeleitet.
Das "BEAM"-Logo oben links ist eine Weiterleitung auf die Startseite
